# X3 Design Project Image Paths

每個室內設計專案使用獨立資料夾，資料夾名稱與 project slug 相同。檔名中的「專案名稱」採用網站 slug，也就是小寫英文與 hyphen，避免空白或中文檔名在部署時產生 URL 編碼問題。

## 命名規則

- `{project-slug}-01.png`: 專案列表封面與 project detail 最大主圖
- `{project-slug}-02.png`: project detail 內頁圖 1
- `{project-slug}-03.png`: project detail 內頁圖 2
- `{project-slug}-04.png`: project detail 內頁圖 3

後續若要替換正式實景照，請維持同樣檔名與副檔名，直接覆蓋檔案即可，不需要修改程式。
